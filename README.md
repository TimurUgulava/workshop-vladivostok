# Воркшоп во Владивостоке

Посадочные страницы воркшопа по сборке AI-ассистентов в Claude Code (Тимур Угулава).

## Страницы

- **День 1** — [`day1/`](day1/) → для собственников и руководителей бизнеса
- **День 2** — [`day2/`](day2/) → для маркетинга и рекламы

Финальный экран каждой посадки — список открытых скиллов с QR-кодом на репозиторий [claude-skills](https://github.com/TimurUgulava/claude-skills).
